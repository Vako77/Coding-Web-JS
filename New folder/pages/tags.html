ძირითადი HTML

თეგი განმარტება
!DOCTYPE განსაზღვრავს დოკუმენტის ტიპს
html განსაზღვრავს HTML დოკუმენტს
head შეიცავს მეტამონაცემებს/ინფორმაციას დოკუმენტისთვის
title განსაზღვრავს დოკუმენტის სათაურს
body განსაზღვრავს დოკუმენტის არსს
h1 to h6 განსაზღვრავს HTML სათაურებს
p განსაზღვრავს აბზაცს გამოიყენება ვრცელი ტექსტებისთვის
br გადავყავართ ახალ აბზაცზე
hr გვიჩენს ავტომატურ ჰორიზანტულ ზოლს

ფორმები და შეყვანა

თეგი განმარტება
form განსაზღვრავს HTML ფორმას მომხმარებლის შეყვანისთვის
input განსაზღვრავს ინფორმაციის შეყვანის კონტროლს
textarea განსაზღვრავს მრავალხაზოვანი შეყვანის კონტროლს (ტექსტის ზონა)
button განსაზღვრავს დაწკაპუნებად ღილაკს
select განსაზღვრავს ჩამოსაშლელ სიას
optgroup განსაზღვრავს დაკავშირებული ვარიანტების ჯგუფს ჩამოსაშლელ სიაში
option განსაზღვრავს ვარიანტს ჩამოსაშლელ სიაში
label განსაზღვრავს ეტიკეტს input ელემენტისთვის
fieldset აჯგუფებს დაკავშირებულ ელემენტებს ფორმაში
legend განსაზღვრავს წარწერას fieldset ელემენტისთვის
detalist განსაზღვრავს წინასწარ განსაზღვრული ვარიანტების სიას შეყვანის კონტროლისთვის
output განსაზღვრავს გაანგარიშების შედეგს

სურათები

თეგი განმარტება
img განსაზღვრავს სურათს
map განსაზღვრავს კლიენტის მხარეს გამოსახულების რუკას
area განსაზღვრავს არეალს გამოსახულების რუქის შიგნით
canvas გამოიყენება გრაფიკის დასახატავად, ფრენის დროს, სკრიპტის საშუალებით (ჩვეულებრივ JavaScript)
figcaption განსაზღვრავს წარწერას figure ელემენტისთვის
figure განსაზღვრავს თვითშეზღუდულ შინაარსს
picture განსაზღვრავს კონტეინერს მრავალი გამოსახულების რესურსისთვის
svg განსაზღვრავს კონტეინერს SVG გრაფიკისთვის
